# study-plan
目前讀書進度與讀書計畫的進度對比：

功能1：寫一個網頁，紀錄當天讀了什麼，可隨時觀看目前進度與幾月幾號(包含幾點幾分)讀了甚麼。

功能2：同上，紀錄自己甚麼時候休息。

可能會配合Todolist的API
